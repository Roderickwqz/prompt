## Canonical Report API
- `generate_model_report_tables(ctx, *, names=None, regime_market_df=None, regime_params=None, incremental_baseline=None, incremental_candidates=None, block_model_map=None, candidate_pool=None) -> dict[str, pd.DataFrame]`
- `generate_model_report_figures(ctx, *, tables=None, regime_map=None) -> dict[str, Figure]`
- `generate_model_report(ctx, **kwargs) -> dict[str, dict]`

## Functional Coverage Matrix
Mandatory analyzer coverage:
- `analyze_quality` -> table: `quality`; figs: ranking bars
- `analyze_stability` -> table: `stability`; figs: worst-window/tail-risk bars
- `analyze_pairwise` -> tables:
  - `pairwise_pred_corr_mean`
  - `pairwise_pred_corr_median`
  - `pairwise_topk_jaccard_mean`
  - `pairwise_error_corr_mean`
  figs: heatmaps
- `analyze_ic_decay` -> table: `ic_decay`; figs: horizon lines/bars
- `analyze_contribution` -> tables: `single_model_ir`, `leave_one_out_ir`; figs: contribution bars
- `analyze_incremental` -> tables: `incremental_delta`, `incremental_leave_one_in`, `incremental_significance`; figs: delta/pvalue views
- `analyze_block_marginal_gain` -> table: `block_marginal_gain`; fig: grouped bars
- `analyze_block_incremental_gain` -> table: `block_incremental_gain`; fig: grouped delta bars
- `analyze_regime` (when `regime_market_df` provided) -> tables: `regime_table`, `regime_map`; figs: regime split comparison
- `analyze_rolling_oos` -> table: `rolling_oos`; figs: rolling metrics
- `analyze_cost_capacity` -> table: `cost_capacity`; figs: net-IR/capacity bars

Control tables:
- `report_status` (per section status)
- `report_meta` (config/names/period summary)

Status enum:
- `ok`
- `skipped_missing_input`
- `empty`
- `error`

## Output Design
Readable report bundle:
- compact numeric tables: sorted; key metrics first
- figure set: small/high-signal; no chart spam
- consistent naming: one key per table/figure
- clear empty/error placeholders in figures
